# Member Images Folder

## Instructions for Adding Member Photos

### Image Naming Convention
Name each image file using the member's name in lowercase with underscores instead of spaces.

**Examples:**
- `elamugil_s.jpg`
- `senthil_kumar_t.jpg`
- `subash_r.jpg`
- `lakshitha_m_g.jpg`

### Image Specifications
- **Format:** JPG, PNG, or WebP
- **Recommended Size:** 400x400 pixels (square)
- **Aspect Ratio:** 1:1 (square)
- **Maximum File Size:** 500KB recommended
- **Background:** Preferably solid color or professional setting

### Fallback
If a member's image is not available, the page will display a default avatar icon with a gradient background.

### How to Add Images
1. Prepare the image (crop to square, resize to 400x400px)
2. Rename the file according to the naming convention above
3. Place the image in this folder (`public/members/`)
4. The website will automatically display the image
